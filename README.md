# Usage
## Setup and compilation

1. Clone repository
    ```bash
    git clone git@github.com:teresa-chow/42-minishell.git minishell
    ```

2. Go inside project directory and run `make`
    ```bash
    cd minishell
    make
    ```

3. Execute `minishell` program
    ```bash
    ./minishell
    ```
